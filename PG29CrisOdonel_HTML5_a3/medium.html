Rules

In this game, the board is divided into cells where mines can randomly appear. In order to win, you have to open all the cells while avoiding all the mines that appear. The way you can know that there if there is a mine near is determined by the number that shows up on the tile.

EX: if 1 shows up on the tile then there is 1 mine next to the tile. Tiles count as being next to each other if they're one tile away from each other. (includes diagonals)

Enjoy.

Controls

Left Click - Reveal Tile
Right Click - Flag Tile
